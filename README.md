# BigCode-codingAnalysis
南京大学软件学院2018级数据科学基础大作业

## 原始数据说明
- user_id 用户唯一标示id
- case-id 题目id
- case-type 题目类型（图结构、字符串、查找算法等8种）
- case-zip 题目包
- upload-id 提交记录id
- upload-time 上传时间
- code-url 对于upload-id所提交的代码
- score 对应upload-id提交得分

## 原始数据文件
- sample.json  10位学生数据样本
- test_data.json  全部271位学生数据样本
